DELVING INTO GREATER DEPTHS

Initially, I wanted to make this game just for the sake of learning, but then the opportunity of this competition struck me. While the game was in its early stages with no artwork done I decided to adapt it to the contest's theme, that being space. So all of the upgrades, all of the monsters, and every fictional element was adapted to fit this theme.

The game you were just introduced to is called Solarium Redemption. It is my first attempt at a real publishable game. With it, I managed to win third place in a game dev contest named FiiCode hosted by the Faculty of Computer Science Iasi. I did all the coding and the artwork for the game and I had a friend sometimes help me with creative ideas and the logo.

While developing this game, I was only able to deepen my knowledge about complex programming challenges, as game development can prove to present pretty unique problems. Some of these complex systems I had to figure out how to build were: infinite map generation, dynamic enemy spawning, a random upgrade system and a bunch of unique weapons, upgrades and so on. It was a pretty hefty learning experience.

Download this game and try it for yourself by clicking here.
(For Windows only)
